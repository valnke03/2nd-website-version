# Personal academic website (static)

## Quick start
- Open `index.html` locally to preview.
- Edit the text directly in the HTML files:
  - `index.html` (home)
  - `research.html` (papers)
  - `projects.html` (projects)
  - `cv.html` (CV summary)
  - `about.html` (bio)
  - `contact.html` (contact + profile links)

## Assets
Place files in `assets/` and link to them from pages.
Current assets:
- `assets/profile.jpg`
- `assets/CV.pdf`
- `assets/master_thesis.pdf`
- `assets/bachelor_thesis.pdf`
- `assets/project_rse.pdf`
- `assets/project_vr_llm.pdf`
- `assets/degree_lu_master.pdf`
- `assets/notebook_nba.ipynb`
- `assets/notebook_abide.ipynb`

## Deploy (GitHub Pages)
1. Create a GitHub repository.
2. Upload the contents of `personal_site_starter/` to the repository root.
3. In GitHub: Settings → Pages → Deploy from a branch → select `main` and `/ (root)`.

## Notes
- The Research and Projects pages include search + tag filters (handled by `script.js`).
- Styling is in `style.css`.
